# 005 - Test

## Objective

Run comprehensive tests to ensure the feature works correctly and code quality is maintained.

## Actions

1. **Test the feature manually** - Verify all functionality works as expected

2. **Check for edge cases** - Test boundary conditions and error scenarios

3. **Run existing test suite** if available (`pnpm test` or similar)

4. **Verify user experience** - Ensure the feature is intuitive and works correctly in the UI

## Output

- Test results summary
- List of any issues found (if any)
- Confirmation that the feature works correctly

## Completion Checklist

Before marking APEX cycle complete, verify:

- [ ] Feature has been manually tested
- [ ] Edge cases have been tested
- [ ] Existing test suite passes (if available)
- [ ] User experience is verified
- [ ] User has verified the feature works

**DO NOT mark cycle complete until ALL items above are verified.**
